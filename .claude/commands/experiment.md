Guide the user through a hands-on experiment using MCP tools. More freeform than a chapter walkthrough — follow their curiosity.

## MCP tools used
- All available MCP tools depending on the experiment

## Parameters
- $ARGUMENTS — what to experiment with (e.g., "matmul shapes", "vanishing gradients", "attention with different temperatures", "what happens when I transpose twice")

## Behavior
1. Set up the experiment using MCP tools
2. Run the first example and explain the result
3. Ask "What do you think will happen if we change X?" — let them predict
4. Run it and compare to their prediction
5. If their prediction was wrong, explain why — this is where real learning happens
6. Suggest 2-3 follow-up experiments they could try
7. Connect the results to the broader concepts from the chapters
